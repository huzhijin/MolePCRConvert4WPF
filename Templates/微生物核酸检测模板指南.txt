# 微生物核酸检测报告模板创建指南

## 模板结构说明

根据提供的截图，此模板应包含以下部分：

1. 报告标题
2. 受检人基本信息
3. 样本信息表格
4. 检测方法
5. 检测阳性病原体列表
6. 报告说明信息

## 注意事项

- 使用`${xxx}`格式替换截图中的`[xxx]`格式占位符
- 在检测阳性病原体列表的起始行添加`[[DataStart]]`标记
- 保存为.xlsx格式

## 详细Excel设计说明

### 1. 表头设计
- A1:G1单元格合并：微生物核酸检测
- A2:G2单元格合并：报告单

### 2. 受检人信息区域
使用以下单元格布局：
```
|            |  姓名:   | ${PatientName} |   性别:  | ${Gender} |       |       |
|            | 送检单位: | ${Department}  |          |           |       |       |
|            | 样本类型: | ${SampleType}  |          |           |       |       |
|            | 检测时间: | ${TestDate}    |          |           |       |       |
```

### 3. 样本详细信息区域
在新表格区域创建以下布局：
```
| *样本信息* |            |              |             |              |       |
|------------|------------|--------------|-------------|--------------|-------|
| 受检人信息 | 姓名:      | ${PatientName} | 性别:      | ${Gender}    |       |
|            | 临床信息:  | ${ClinicalInfo} |           |              |       |
|            | 病历号:    | ${PatientId}   |            |              |       |
|------------|------------|--------------|-------------|--------------|-------|
| 样本信息   | 样本条码:  | ${SampleId}    | 实验编号:  | ${ExperimentId} |    |
|            | 接收日期:  | ${CollectionDate} | 采样日期: | ${TestDate}  |       |
|            | 样本类型:  | ${SampleType}   | 样本条码:  | ${SampleCode} |     |
|------------|------------|--------------|-------------|--------------|-------|
| 送检方信息 | 送检单位:  | ${Department}   | 送检科室:  | ${Section}   |       |
|            | 送检医生:  | ${Doctor}      |            |              |       |
```

### 4. 检测方法区域
```
| *检测方法* |                          |       |       |
|------------|--------------------------|-------|-------|
|            | 实时荧光定量PCR法        |       |       |
```

### 5. 检测阳性病原体列表
在这个表格的第一行数据单元格放置`[[DataStart]]`标记：
```
| *检测阳性病原体列表* |                |       |       |
|----------------------|----------------|-------|-------|
| 病原体名             | 浓度 (拷贝/mL) | CT值  | 备注  |
| [[DataStart]]        |                |       |       |
```

### 6. 报告说明信息
在表格下方添加说明文本：
```
1. 本试剂盒的检测结果仅供临床参考，不得单独作为患者管理的依据，检测结果应结合相关医学信息进行综合分析。
2. 该适合类型及感染的最高浓度与特定病原微生物在的时间未经验证，因此，对同一患者进行多次、多部位采集样本能降低阴性结果的可能。
3. 样本检测结果和样本收集、处理、运送及保存质量有关，其中任何环节的错误可能导致假阴性结果。
4. 如果样本处理时没有控制交叉污染，可能出现假阳性结果。
5. 检测靶标的变异也可能会导致假阴性结果的出现。
6. 阴性检测结果表明样本中病毒滴度低于本试剂盒的最低检出限，不能排除病原微生物感染的可能性;同时，阴性的检测结果也不能排除检测范围之外的微生物感染的可能。
```

## 保存说明
将此Excel模板保存为"微生物核酸检测报告模板.xlsx"到Templates目录。使用时，系统会自动替换所有占位符并在[[DataStart]]位置填充实际检测数据。
